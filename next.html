Learning Web Design

Once you master the basics of authoring HTML, you’ll need to focus on what your next steps will be. Web design is a pretty open-ended term, and there are many paths you can take.

Learn how the web works.
It’s always amazing how many web designers focus on learning just what they need to know to complete a project.

Determine an area of focus, and start there.
There are a lot of areas of web design that you could specialize in, so pick one and begin learning deeply.

Never stop learning.
The web is constantly evolving. Even if you become an expert, there is always something new to learn.

Break things!
There is no better teacher than experience. Build and experiment constantly.
