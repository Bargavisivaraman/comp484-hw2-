A markup language

Because HTML is often a personâ€™s first experience with coding, there can be some apprehension about learning it. Thankfully, HTML syntax is relatively simple and easy to learn.

HTML is a markup language. Content on the page is marked up using tags that identify the content inside them. 

<p>This is a paragraph.</p>

Tags consist of a left-angle bracket (<), a tag name, and a right-angle bracket (>). Closing tags include a forward slash.

Basic document structure

All HTML documents share a basic structure consisting of the <html>, <head>, and <body> elements. 


<html>
  <head>
  </head>
  <body>
  </body>
</html>

The head contains metadata and references to external resources, while the body contains the visible content of the page.

Doctypes

A doctype declaration tells the browser which version of HTML is being used and helps ensure standards-compliant rendering. 

<!DOCTYPE HTML>

Element attributes

Some elements can be enhanced through attributes. Attributes are added to the opening tag and consist of a name and value. 

<h1 class="headline">Main heading</h1>

Replaced elements

Replaced elements represent content that is replaced by an external resource such as an image or video. 

<img src="photo.jpg" alt="Sample image">

Code structure

HTML documents create structure by nesting elements inside one another. Child elements must be closed before parent elements. 

<p>Incorrect <strong>example</p></strong>
<p>Correct <strong>example</strong></p>

Commenting code

Comments are used to leave notes in the code and are ignored by the browser. 

<!-- This is a comment -->

Using special characters

Reserved characters can be displayed using named character entities, such as &amp; for an ampersand.

You can find more examples on the HTML Reference page.
